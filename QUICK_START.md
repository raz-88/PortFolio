### Edit Content

Each component is independent and easy to edit:

1. **Home page** → `src/components/Home/Home.jsx`
2. **About page** → `src/components/About/About.jsx`
3. **Add colors** → `src/styles/global.css`
4. **Component styling** → Individual CSS files

### Build for Production

```bash
npm run build
```

This creates a `dist` folder with optimized files ready for deployment.

### Deployment

Choose any platform:
- **Vercel** (easiest for React)
- **Netlify**
- **GitHub Pages**
- **Any hosting with Node.js**
